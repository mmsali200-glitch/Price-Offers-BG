## 🏢 المُوديول الثالث — الشركات المتعددة (Multi-Company)

## 🧩 ملاحظات تكاملية للموديولات الثلاثة

### 🔗 العلاقات بين الموديولات

| الموديول | يتطلب | يتكامل مع |
|---------|------|----------|
| 🚗 الأسطول | لا شيء (مستقل) | HR، المشاريع، المحاسبة |
| 🏗️ المقاولات | المشاريع + المخازن + المشتريات + المحاسبة | الأسطول، HR، التصنيع |
| 🏢 الشركات المتعددة | المحاسبة | **كل** الموديولات الأخرى |

### 📅 ترتيب التطبيق المقترح

```
المرحلة 1 (التحليل والإعداد):
  → الشركات المتعددة (إذا مطلوب) — يجب أولاً

المرحلة 2 (التطوير الأساسي):
  → المحاسبة + المخازن + المشتريات + المبيعات
  → الأسطول (إذا مطلوب — سهل التطبيق)

المرحلة 3 (التطوير المتقدم):
  → المقاولات (إذا مطلوب — يحتاج تخصيصات)
  → التكاملات بين الموديولات

المرحلة 4 (الإطلاق والدعم):
  → الذهاب الحي + التدريب + الدعم
```

## 📋 قائمة تحقق سريعة قبل تضمين أي من هذه الموديولات

### للأسطول:
- [ ] العميل لديه أكثر من 3 مركبات؟
- [ ] هل تكاليف الوقود/الصيانة جزء من هيكل التكاليف؟
- [ ] هل هناك سائقين موظفين أم سائقي مهام؟

### للمقاولات:
- [ ] العميل شركة مقاولات (وليس فقط لديها مشاريع داخلية)؟
- [ ] هل يستخدم BOQ في عروضه؟
- [ ] هل يصدر فواتير مرحلية مع احتجاز؟
- [ ] هل لديه مقاولين فرعيين؟
- [ ] هل يحتاج تتبع التقدم بالنسب المئوية؟

### للشركات المتعددة:
- [ ] لدى العميل أكثر من سجل تجاري؟
- [ ] هل توجد عمليات بيع/شراء بين الشركات؟
- [ ] هل يحتاج تقرير مدمج للمجموعة؟
- [ ] هل تختلف العملات بين الشركات؟
